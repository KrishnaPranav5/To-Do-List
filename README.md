# 📝 To-Do List Application

A simple command-line To-Do List application built using Python. This project helps users manage their daily tasks efficiently.

## 🚀 Features
- Add new tasks
- View all tasks
- Update task details
- Delete tasks
- Store tasks in a JSON file

## 🛠️ Technologies Used
- Python
- JSON (for data storage)
